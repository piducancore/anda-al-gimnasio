# 🏋️ anda-al-gimnasio

Esta es una función _serverless_ que se ejecuta todos los días para reservarme un cupo en el gimnasio y avisarme vía [Telegram](https://core.telegram.org/). Hecho con [Netlify Scheduled Functions](https://docs.netlify.com/netlify-labs/experimental-features/scheduled-functions/).
